## About the image

Node Docker image to launch Javascript, React apps to GCP (GCloud)

Core image:
 - gcr.io/gcp-runtimes/ubuntu_16_0_4:latest

Contains:
 - Node 12.x
 - Yarn
 - Google Cloud SDK


## Usage

Use it with `circleCI` : ` - image: alsco77/node-gcp-dockerfile:origin`
